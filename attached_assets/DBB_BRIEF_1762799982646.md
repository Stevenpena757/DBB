## Purpose
DallasBeautyBook is a mobile-first directory + social platform for DFW health, beauty, and aesthetics.

## Core User Goals
- Find trusted providers by city/service; evaluate via photos/reviews; save/follow.
- Owners claim profile, post updates, run promos, see ROI analytics.

## Brand — Soft Copper Radiance 2.0
- Gradient: linear-gradient(135deg, #EFC6B3 0%, #F5DFC9 40%, #FFF9F5 100%)
- Colors: Cream Ivory #FFF9F5, Champagne Sand #F7E9D8, Soft Copper #DFA58A,
  Rose Gold #EAC3B6, Warm Taupe #B99E8A, Cocoa Ink #3D2B1F
- Support: Highlight Gold #F2C97D, Soft Shadow rgba(61,43,31,.08)

## Typography
- Cormorant Garamond (headers), DM Sans (body), Outfit (UI).

## Layout
- 16px rounded cards, soft gradients, large breathing room, consistent icon style.

## Key Screens
- Homepage, Business page, Mobile bottom nav (Home, Explore, Community, Saved, Profile).
