STEP 2 — Fonts
--------------

Use Google Fonts:
- Headings: Montserrat
- Body: Source Sans Pro
- UI / Labels: Work Sans

Add the following to global CSS:

:root {
  --font-heading: 'Montserrat', sans-serif;
  --font-body: 'Source Sans Pro', sans-serif;
  --font-ui: 'Work Sans', sans-serif;
}

h1, h2, h3, .heading {
  font-family: var(--font-heading);
}
body, p {
  font-family: var(--font-body);
}
.ui-label, .chip, .nav-item {
  font-family: var(--font-ui);
}


STEP 3 — UI Component Styling Rules
-----------------------------------

Apply these UI rules across the site:

- Cards:
  - Rounded 18px
  - Soft shadows
  - Ivory/white backgrounds

- Buttons:
  - Primary uses Emerald
  - Secondary uses Dusty Rose / Champagne
  - Rounded 12px
  - Soft drop shadows

- Background:
  - Use Blush Base (#F4ECE9)
  - Avoid pure white for full-page backgrounds

- Gradients:
  - Use botanical gradients sparingly:
    --gradient-botanical
    --gradient-emerald-rose

- Icons:
  - Outline style
  - Rounded corners
  - Stroke 1.5px
